# DSL-парсер запросов «Библиотека»
### Описание
Реализация предметно-ориентированного языка (DSL) для формирования запросов к базе данных «Библиотека» и синтаксического анализатора на основе формальной грамматики. Разбор выполняется методом рекурсивного спуска.

### Пример запросов
* найди все книги
* покажи статьи 2020 года
* *выведи журналы по программированию

### Запуск
python main.py
